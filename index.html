PDX Transit

PDX Transit is a project that I've been working on since 2015. Since then, the app has grown and is now available on Apple Watch and Macintosh. It uses Trimet's open data API to help users locate nearby transit stops, get arrival times, and service alerts for public transit in Portland, OR. Download it today from the App Store for free!
