üõ† Technical stuff for the nerds

Music Widget uses Scripting Bridge and Distributed Notification Center to send and receive commands with Apple Music on macOS. 99%** of the UI is written in Core Graphics and Core Animation to achieve crisp UI that is also optimized for non-Retina Displays. Having said that, Music Widget is completely a native app; written completely with Swift and AppKit.

Requires macOS Big Sur and Apple Music. Does not work with Spotify on macOS.
