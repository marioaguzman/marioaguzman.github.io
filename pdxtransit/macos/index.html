PDX Transit for macOS

Description Here

Nearby Transit Stops
Uses your current location to get nearby transit stops for TriMet.
Arrival Times
Select a nearby transit stop or look up a transit stop via Stop ID to get its arrival times.
Service Alerts
Get all published service alerts by TriMet organized by transit line.
Bookmarks

CloudKit

Bookmark your favorite transit stops for quick access. Bookmarks store on iCloud and sync across all your iOS, watchOS, and macOS devices.
Dark Mode

AppKit

PDX Transit for macOS includes the dramatic dark interface for Dark Mode.
Widget

Widget

PDX Transit includes a Today View Widget with quick access to your bookmarked transit stops.
Help
PDX Transit for macOS includes a proper classic Mac OS X Help Book that can be accessed from the Help menu and is available offline.
Spotlight
Previously selected or viewed transit stops get index with Spotlight so they appear in Spotlight search results including a preview with real-time arrivals.
