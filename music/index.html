Companion apps for
Apple Music on macOS

Tiny apps. Huge personalities. Classic design.

macOS 26 Tahoe Support

With the release of macOS Tahoe, Music v1.6.0 no longer supplies the data required for these Music Companion Apps to operate correctly. Unfortunately, there is no technical workaround for this change. As a result, macOS Sequoia remains the last version fully supported by QuickTune, Music Remote, Music MiniPlayer, and Music Widget.

QuickTune 7

Control Apple Music with this simplified controller that has the look and feel of QuickTime 7.
Learn more about QuickTune



Music Remote

Go back to the first Mac OS X Public Beta with this replica of the short lived "Music Player." Now Available.



Music Widget

A replica of the original iTunes widget introduced with Dashboard in Mac OS X Tiger, this small remote can be used to control playback, volume, and playlists.
Learn more about Music Widget

Music MiniPlayer

A replica of the original iTunes 10 MiniPlayer, this small remote can be used to control playback, volume, and playlists; all with an interface that is sure to take you back in time.
Learn more about Music MiniPlayer
